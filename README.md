# Simulador de Editor de Texto (Undo/Redo) con Pilas 

## Objetivo del Proyecto
El objetivo principal de este proyecto es comprender y aplicar la estructura de datos "Pila" (Stack) mediante su implementación manual en Java. Se utiliza esta estructura para crear un simulador de un editor de texto básico en consola que soporta las funciones de Deshacer (Undo) y Rehacer (Redo), trabajando de manera colaborativa.

## Instrucciones de Ejecución
1. Asegúrate de tener instalado Java (JDK 8 o superior).
2. Clona este repositorio en tu máquina local:
   git clone https://github.com/JuanRojas322/Simulador-Pilas.git
3. Navega hasta la carpeta del proyecto.
4. Compila los archivos Java:
   javac SimuladorEditor.java EditorTexto.java PilaPersonalizada.java
5. Ejecuta el programa:
   java SimuladorEditor
